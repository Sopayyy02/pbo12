Proyek ini merupakan implementasi dari tugas mata kuliah **Pemrograman Berbasis Objek (PBO)** menggunakan **Java Swing** dan **Java Persistence API (JPA)**.  
Aplikasi ini dirancang untuk mengelola **beberapa entitas data** (misalnya Buku dan Penerbit) melalui antarmuka GUI berbasis **Tab Container (JTabbedPane)** serta menyimpannya secara permanen di **database PostgreSQL**.  

Selain fitur CRUD (Create, Read, Update, Delete) dasar, aplikasi ini juga memiliki kemampuan **mengimpor data dari file CSV**, sehingga pengguna dapat menambahkan data dalam jumlah besar secara otomatis.
